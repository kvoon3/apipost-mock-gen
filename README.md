# apipost-mock-gen

Automatically generate mock api from [apipost](https://www.apipost.cn/apidoc/) exported documents

## Usage

```sh
ni -g apipost-mock-gen
```

```sh
apipost-mock-gen [input] --output <output> --parser <parser> --template <template>
```

## Parsers

- apipost-markdown

## templates

- vite-plugin-mock
